Class WinnerController

Controller for the /api/winner route Given a 3x3 tic-tac-toe board, it will return the winner (if there is one) from the state of the board. Player.Tie (0) = Cat's game (tie) Player.User (1) = X won Player.AI (2) = O won Player.None (-1) = Board is incomplete
